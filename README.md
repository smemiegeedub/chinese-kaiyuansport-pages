# chinese-kaiyuansport-pages

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容涉及中国开源运动相关的信息、资料与展示，以静态页面的形式进行存储和托管。仓库本身不针对任何特定域名或网站，仅作为页面资源的集合与版本管理使用。

## 目录说明

```
.
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式表、脚本等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本说明文件
```

- `pages/` 目录下每个子文件夹或文件对应一个归档页面，命名遵循内容主题的简要英文或拼音标识。
- `assets/` 用于统一存放各页面共享或专用的资源，避免重复引用。

## 页面归档说明

- 每个 HTML 页面均为独立完整的静态文件，可直接在浏览器中打开查看。
- 页面内容主要围绕中国开源运动的相关议题、活动记录、资料整理等，不涉及具体商业网站或推广信息。
- 归档遵循“一页一事”的原则，便于维护和引用。
- 所有页面均保持原始结构，不添加额外跟踪或统计代码。

## 维护说明

- 欢迎通过 Pull Request 提交新的归档页面或修正现有页面内容。
- 提交前请确保页面为独立 HTML，不依赖外部不可控资源（除非资源本身已稳定存档）。
- 建议在 `pages/` 下新建文件夹，并将页面与配套资源放在同一文件夹内，避免路径混乱。
- 如更新现有页面，请保留修改记录，并在 PR 描述中简要说明变更原因。
- 本仓库不主动推送或推广任何页面内容，仅作为资料存档使用。
